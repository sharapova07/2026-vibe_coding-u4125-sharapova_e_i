# Ally For Projects — Telegram-бот

Бот на **Python** и **python-telegram-bot**: расписание встреч, роли команды, напоминания за 15 минут до начала (часовой пояс **Europe/Moscow**).

## Требования

- Python 3.9+ (рекомендуется 3.10+)
- Токен бота от [@BotFather](https://t.me/BotFather)

## Установка

```bash
cd ally-for-projects-bot
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## Конфигурация

1. Скопируйте пример переменных окружения:

   ```bash
   cp .env.example .env
   ```

2. Откройте `.env` и укажите `BOT_TOKEN`.

## Запуск

```bash
source .venv/bin/activate
python bot.py
```

Данные событий и авторизаций хранятся в SQLite (файл `data/bot.sqlite3` по умолчанию). Каталог `data/` создаётся автоматически.

## Команды (кратко)

| Команда | Кто |
|--------|-----|
| `/start` | Все: приветствие и авторизация по ФИО |
| `/commands` | Авторизованные: список команд |
| `/schedule` | Авторизованные: сегодня / завтра / неделя |
| `/event <id>` | Авторизованные: карточка события |
| `/add_event`, `/change_event`, `/del_event`, `/list_events` | Только руководитель и администратор проекта |

Список ФИО и ролей задан в коде `bot.py` (константа `TEAM_ROSTER`).

## Примечания

- Напоминания получают участники события, у которых есть **совпадающее ФИО** в базе авторизованных пользователей (после `/start`).
- После перезапуска бота отложенные напоминания подхватываются из базы заново.
